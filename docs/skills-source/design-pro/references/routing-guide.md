# Routing Guide

## Quick Decision Rules

- Choose system-driven for design systems, admin panels, dashboards, tables, forms, accessibility, consistency, and reusable components.
- Choose creative-driven for brand-forward UI, landing pages, campaigns, differentiated visuals, motion-heavy surfaces, and expressive layouts.
- Choose hybrid when the request explicitly asks for both professionalism and personality, or when product UI needs both trust and distinction.

## Default Rule

If the request mixes structure and expression, use hybrid.
