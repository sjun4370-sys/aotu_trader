# Skills Source

This directory is the canonical source for the personal frontend design skills:

- `ui-ux-pro-max`
- `frontend-design`
- `design-pro`

## Sync Targets

- Claude: `C:\Users\A\.claude\skills\`
- OpenCode: `C:\Users\A\.config\opencode\skills\`

## Update Workflow

1. Edit the canonical files here first.
2. Mirror the updated directories into both targets.
3. Restart Claude/OpenCode if discovery is stale.
4. Verify each skill can be discovered and loaded.

## Rule

Do not hand-edit the mirrored copies first. Source-of-truth changes belong in this directory.
